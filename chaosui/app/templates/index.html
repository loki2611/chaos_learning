Setup Guide

How to deploy the Chaos UI on Kubernetes alongside TradeSphere

Step-by-Step Deployment
Local Quick Start (no Kubernetes)
# 1. Install dependencies
pip install flask requests chaostoolkit \
    chaostoolkit-kubernetes

# 2. Set target URL (your running TradeSphere)
export TARGET_APP_URL=http://localhost:5000
export APP_NAMESPACE=tradesphere

# 3. Start Chaos UI
python run.py
# Opens at http://localhost:8090

# 4. In another terminal, keep port-forward alive
kubectl port-forward service/tradesphere-svc \
    5000:80 -n tradesphere --pod-running-timeout=5m &
